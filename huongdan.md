trước tiên cần phải cd backend rồi npm run dev ở backend trước sau đó mới chạy npm run dev ở frontend nhé, nếu không sẽ lỗi (khởi động dữ liệu ở server )

phía fe:
khi sử dụng lệnh git clone https://github.com/votruongthinh/FE_GroceryStore.git chuyển từ sang file frontend bằng lệnh cd frontend, sau đó chạy lệnh npm i ở terminal để tải thử viện node_modules để chạy toàn bộ dự án , nếu không có node_modules hoặc chạy lần đầu sẽ lỗi nhé, sau đó cd frontend rồi npm run dev ở file package.json ở phần script "dev" nhé (sẽ hiện port 5173)
